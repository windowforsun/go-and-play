# 고 앤 플레이 (go-and-play)

사다리 타기·룰렛·손가락 짚기 등 **내기하거나 술래를 정하는 게임 모음**.
브라우저에서 돌아가는 정적 웹앱으로, GitHub Pages에 배포한다.

- 기술 스택: **순수 HTML / CSS / JS (ES 모듈, 빌드 단계 없음)**
- 구조: **허브 + 게임 모듈** (게임을 하나씩 붙여나감)
- 온라인(멀티플레이)은 이후 외부 실시간 서비스로 확장 가능 (프론트는 그대로 유지)

## 📁 디렉터리 구조

```
go-and-play/
├── README.md
├── brainstorm/          ⚠️ 브레인스토밍 전용 — 정식 코드 아님 (버려질 수 있음)
│   ├── README.md
│   └── concepts/        디자인 컨셉 목업 (A/B/C 비교용)
└── (정식 앱 코드는 설계 확정 후 추가)
```

> ⚠️ **중요**: `brainstorm/` 아래 파일은 **아이디어 검토용 목업**이며 **정식 코드가 아니다.**
> 정식 앱 코드와 절대 섞지 않는다. 자세한 내용은 [`brainstorm/README.md`](./brainstorm/README.md) 참고.
