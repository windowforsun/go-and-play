# ⚠️ brainstorm/ — 브레인스토밍 전용 (정식 코드 아님)

이 폴더 안의 모든 파일은 **아이디어를 눈으로 확인하기 위한 임시 목업**이다.

- ❌ **정식 앱 코드가 아니다.** 여기 있는 HTML/CSS를 그대로 제품에 쓰지 않는다.
- ❌ 컴포넌트 재사용, import 대상이 아니다.
- ✅ 디자인 컨셉·레이아웃·흐름을 **비교하고 결정**하기 위한 용도일 뿐이다.
- ✅ 결정이 끝나면 이 폴더는 통째로 삭제되거나 아카이브될 수 있다.

정식 앱 코드는 저장소 루트(또는 `src/`)에 별도로 만들어진다. **절대 섞지 않는다.**

## 내용물

### `concepts/` — 디자인 컨셉 목업 (택1 하기 위한 비교)

| 파일 | 컨셉 | 한 줄 설명 |
|------|------|-----------|
| `index.html` | 비교 허브 | 세 컨셉을 한눈에 보고 이동 |
| `concept-a-pixel.html` | **A. 도트/픽셀** | 8비트 레트로, 통통 튀는 픽셀 캐릭터 |
| `concept-b-handdrawn.html` | **B. 손그림** | 공책에 손으로 슥슥 그린 낙서 느낌 |
| `concept-c-jelly.html` | **C. 젤리 팝 (라이트)** | 말랑말랑 파스텔, 세련된 모바일 미니게임 감성 |
| `concept-c-dark.html` | **⭐ C-다크 (현재 진행 방향)** | C 발전형 — 다크 배경 + 대표 마스코트 "말랑이" + 애니메이션 |
